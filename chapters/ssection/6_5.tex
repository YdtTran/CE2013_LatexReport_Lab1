\begin{enumerate}
    \renewcommand{\labelenumi}{\alph{enumi}.}
    \item Find the required value for the resistor.
              {
                  By applying Ohm's law for $3.9k\Omega$ resistor, we have:
                  \vspace{-0.15cm}
                  \[
                      I = I_1 = \dfrac{5}{3.9} = 1.282 (mA)
                  \]
                  Then by applying Ohm's law for resistor $R$, we have:
                  \vspace{-0.15cm}
                  \[
                      R = \dfrac{V}{I} = \dfrac{24-5}{1.282} = 14.821 (k\Omega)
                  \]
              }
    \item Use Table 2.1 in the lecture slide to select a standard 10\% tolerance resistor for $R$. $R$ in the circuit may be a single resistor or a combination of resistors as long as these resistors meet the standard values and are available in the market.
    \item Using the resistor selected in (b), determine the voltage across the $3.9\ \text{k}\Omega$ resistor.
    \item Calculate the percent error in the voltage $V_1$ if the standard resistor selected in (b) is used.
    \item Determine the power rating for this standard component.
\end{enumerate}
